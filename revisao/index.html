Revisão Geral

A tag <DOCTYPE html> é responsável por identificar um arquivo como HTML 5

A tag <html lang=""> é responsável por definir a linguagem padrão da página ao abrir

A tag <meta charset="UTF-8"> é responsável por permitir que a página consiga ter caracteres especiais e pontuações

A tag <meta name="viewport" content="width=device-width, initial-scale=1.0"> é responsável por ajustar o tamanho do preenchimento da página

A tag <title> é responsável por informar o título que fica na página do navegador(nome da aba dentro do navegador)

A tag <head> é responsável por fazer o cabeçalho da página

A tag <body> é responsável por fazer o corpo da página

A tag <p> é responsável por fazer os parágrafos da página

A tag <h1> até <h6> é responsável por informar o nível dos títulos, sendo títulos de nível 1 ao 6, o nível 1 é o principal e com as maiores letras, os níveis seguintes são tópicos do nível anterior
EX: <h2> é subnível do <h1>

A tag <hr> é responsável por colocar uma linha horizontal logo a abaixo do parágrafo ao qual foi inserido

A tag <br> é responsável por realizar uma quebra de linha no parágrafo ao qual foi informado

A tag <img src="" alt=""> é responsável por importar a imagem para a página a parte alt="" é responsável por informar a finalidade da imagem dentro do código

A tag <link:favicon> é responsável por importar o favicon da página o favicon é o ícone que fica do lado do título da página no navegador
EX: O google, o favicon do google é o G que fica do lado nome google

A tag <address> é responsável por um destaque a um endereço informado dentro da página

A tag <strong> é responsável por deixar as palavras da página em negrito

A tag <em> é responsável por deixar as palavras da página em itálico

A tag <mark> é responsável por realizar uma marcação na página, a tag mark não é reconhecida pelo visual studio cod, tem que realizar um encapsulamento da parte do parágrafo para que possa informar o mark, para realizar o encapsulamento tem que dar ctrl+shift+p

A tag <small> é responsável por deixar partes do parágrafo do com a letra menor

A tag <del> é responsável por deixar algumas partes do parágafro com uma linha cortando, serve para informar ao leitor que pode ler o que está alí, mas não é importante

A tag <ins> é responsável por deixar uma linha embaixo de algumas palavras dentro do parágrafo, servindo para informar ao leitor que tem de ser dada uma maior atenção a essa parte do parágrafo

A tag <sup> é responsável por jogar um determinado termo dentro do parágrafo acima do padrão das outras palavras dentro do parágrafo

A tag <code> é responsável por deixar todos os caracteres dentro da tag com a mesma largura

EX: apenas para demonstração

A tag <pre> é responsável por apresentar o que foi escrito dentro da tag justamente da maneira a qual está aparecendo

EX: 

apenas
        para demonstração
        do que essa tag é capaz

A tag <q> é responsável por apresentar citações(coisas ditas em livros ou por alguém em um momento registrado)

EX: Apenas para demonstração

A tag <blockquote> é responsável por pegar grandes citações

EX:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestias exercitationem dolor atque iusto eum hic maxime, voluptas, veritatis, inventore nihil quam delectus iure fugiat. Excepturi veniam quae modi impedit quisquam!

A tag <abbr> é responsável por apresentar a descrição das abreviações que estão sendo mostradas

EX:HTML

A tag<bdo dir ="rtl"> é responsável por inverter a ordem do texto informada dentro da tag

EX: apenas para demonstração

A tag <ol> é responsável por criar uma lista ordenada

A tag <li> é responsável por fazer referência a um item da lista

A tag <ul> é responsável por criar uma lista não ordenada

A tag <dl> é responsável por criar uma lista de definição

A tag <dt> é responsável por informar o termo na lista de definição

A tag <dd> é responsável por infotmar a descirção

A tag <a href=""> é responsável por criar um link que irá transitar entre as páginas do site ou por abrir um link externo

A tag <picture> é responsável por pegar diversos arquivos e selecionar qual deles é o mais apropriado para aparecer na imagem

A tag <source> é responsável por informar os parâmetros aos quais a partir de qual momento a imagem informada dentro da tag picture deve aparecer

A tag <audio src=""> é responsável por importar o áudio para dentro da página a qual está fazendo parte

A tag <video> é responsável por importar um vídeo para o site ou trazer a url do video para dentro do site

A tag <button> é responsável por criar um butão dentro da página a qual estamos
